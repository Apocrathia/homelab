# Infrastructure Components

This directory contains core infrastructure components required for cluster operation.

## Typical Components

- Cert-manager
- Ingress controllers
- Storage classes
- Monitoring stack
- Logging stack
- Network policies
- 1Password Connect Operator

## Guidelines

- Define clear dependencies between components
- Use proper resource requests/limits
- Implement appropriate RBAC
- Configure monitoring and alerts
- Follow security best practices
- Use 1Password for secrets management
